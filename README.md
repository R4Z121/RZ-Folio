## Getting Started

Here are steps to install and run this project to your computer locally :

### Prerequisites

1. Node.Js

    ```
      node --version
      npm --version
    ```
2. Git

   ```
     npm install git
     git --version
    ```
### Installation

1. Clone the repo

    ```
      git clone https://github.com/R4Z121/RZ-Folio.git
    ```
2. Install NPM Packages

    ```
      npm install
    ```
3. Run project in development mode

    ```
      npm run dev
    ```
4. Build Project

    ```
      npm run build
    ```
